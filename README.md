# video-upload-processor

Create a webpage where users can upload videos and preview them on the page.
After clicking the upload button, the video will be sent to the backend.
The backend can edit the uploaded video and send the edited content back to the webpage for the user to download.
During the editing process, a processing status should be displayed.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/video-upload-processor.git
cd video-upload-processor
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
